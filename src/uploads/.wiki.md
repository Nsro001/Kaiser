Project Summary
CotizaciónWeb is an advanced quotation management system aimed at optimizing the creation and management of quotations, purchase orders, and client relations. Utilizing modern web technologies, it features a streamlined interface that enhances user experience by allowing efficient tracking of quotations, comprehensive client management, and insightful metric visualization.

Project Module Description
Dashboard: Displays key metrics and visualizations related to quotations.
Cotizaciones: Lists all quotations with filtering capabilities.
Crear Cotización: A form for creating new quotations, enabling client and currency configurations.
Clientes: Manages client information, including creation, listing, and editing.
Servicios/Productos: Handles the management of products and services, including creation and listing.
Sidebar: Facilitates navigation between different sections of the application.
Header: Contains search functionality and user options.
